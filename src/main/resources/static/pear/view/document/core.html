方法 描述 参数 备注
admin.collapseSide 侧边收缩 无 无模式限制
admin.refreshThis 刷新当前 无 无模式限制
admin.refresh 刷新指定 id: 编号 仅限 TAB 模式
admin.addTab 新增卡片 id:编号,title:标题,url:路径 仅限 TAB 模式
admin.changeTabTitle 修改标题 id:编号,title:标题 仅限 TAB 模式
admin.closeTab 关闭卡片 id:编号 仅限 TAB 模式
admin.closeCurrentTab 关闭当前 无 仅限 TAB 模式
admin.closeAllTab 关闭所有 无 仅限 TAB 模式
admin.closeOtherTab 关闭其他 无 仅限 TAB 模式
admin.changeIframe 切换页面 id:编号,title:标题,url:路径 仅限 IFRAME 模式
admin.jump 兼容切换 id:编号,title:标题,url:路径 无模式限制
admin.fullScreen 全屏模式 无 无模式限制
